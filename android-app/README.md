# Android app

Initial Kotlin Android skeleton for the Samsung Galaxy S20 robot interface.

Current milestone:

- dashboard UI;
- MVVM `RobotDashboardViewModel`;
- mock BLE repository;
- ESP32 telemetry model;
- structured robot commands;
- command validator;
- text-only privacy wrapper;
- mock assistant client;
- placeholder text-to-speech service.

Movement remains simulation-only. There is no real motor control in this app.

Open in Android Studio from this `android-app` directory.
